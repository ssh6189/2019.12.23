모델로부터의 독립변수와 종속변수가 존재 -> 모델이 예측한 값을 생성하는것 : predict

오차를 가지고 정확도 측정



선형관계가 아니라, 곡선형태로 나타날시, 그럴때는 다른 방법 : 다항회귀분석

단항회귀분석 : y = ax + b
다항회귀분석 : y = ax^2 + ax + b

Polynomail Feature : 데이터에 대해서 다항식으로 변형

score() : r-squre[결정계수], 얼마나 잘 만들어졌는가

바이어스 : 치우침 -> 종속변수에 영향을 주는 독립변수가 하나가 아니다.



그래서, 여러개의 독립변수에 관해서 종속변수와의 관계를 찾는것이 다중선형회귀분석

어떤 방법이든, 객체 생성은 늘 LinearRegression()이다.

변수의 유형 : 범주형, 서열, 등간, 비용

등간과 비용은 연속형 변수이다.

회귀분석에서 중요한것 : 종속변수가 연수형이어야 한다. 0아니면, 1인 범주형 변수면, 또는 0, 1, 2, 3인 다항인 경우에는 로지스틱 회귀분석, bool도 포함

p-value : 유의값 0.05
